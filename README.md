# 🌐 Portfolio Website  

A modern, responsive personal portfolio website built with **HTML, CSS, and JavaScript**.  
This project is designed to showcase skills, projects, and achievements in a visually appealing and interactive way.  

---

## 🚀 Features  
- Responsive design (works on all screen sizes)  
- Smooth animations and transitions  
- Typing effect for dynamic text introduction  
- Social media integration (Facebook, Instagram, LinkedIn, X)  
- Glowing buttons, hover effects, and modern UI styling  

---

## 🛠️ Technologies Used  
- **HTML5** – Structure and content  
- **CSS3** – Styling, gradients, and animations  
- **JavaScript (ES6)** – Interactive typing effect & dynamic behavior  

---

## 📂 Project Structure  
├── index.html # Main HTML file
├── style.css # Styles and animations
├── script.js # JavaScript for typing effect and interactivity
└── images/ # Profile and project images
